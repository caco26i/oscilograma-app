## Features

- [X] Posts
- [X] Pages
- [X] Custom posts
- [X] Authors
- [X] Categories
- [X] Tags
- [X] Custom taxonomies
- [X] Search (Posts, Pages & Custom)
- [X] 10+ languages
- [X] Offline
- [X] Syntax highlighter
- [X] Social sharing
- [X] Docker build
- [X] Admob support
- [ ] App rate plugin
- [ ] Comments
- [ ] Push Notifications
- [ ] Login via JWT
